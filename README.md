## 📝 About the Project

The **Stock Market Dashboard** is an **interactive web application** built using **Streamlit**.  
It allows users to:  

✨ **Track real-time stock prices**  
💼 **Manage and monitor personal portfolios**  
📈 **Visualize stock trends with candlestick charts**  
📰 **Read the latest financial and market news**  

Perfect for **students, beginners, and investors** who want a **visual and user-friendly stock monitoring tool**.

---

## 💡 Features

### 📊 Market Overview
- 💰 **Latest, High, and Low Prices** of any stock  
- 📉 **Candlestick charts** to track trends  
- 🔮 **Next-day price prediction**  

### 💼 Portfolio Tracker
- ➕ Add stocks with number of shares  
- 📊 Track **portfolio value** and performance  
- 🗃️ Clean **data table view**  

### 📰 News Section
- 🗞️ Get **latest market news**  
- 🔗 Clickable links to **full articles**  

### 👤 Profile Section
- 🧑‍💻 Shows **user info** and project details  

### 🌙 Dark Mode UI
- 🖤 Modern **dark-themed interface**  
- 🎨 Custom **buttons, input fields, and cards**  

---

## ⚙️ Technologies Used

- 🐍 **Python 3.x**  
- 🌐 **Streamlit** – Web application framework  
- 📈 **Plotly** – Candlestick & trend charts  
- 🗃️ **Pandas** – Data handling & analysis  
- 📊 **YFinance / APIs** – Real stock data (optional)  
- 🎨 **HTML & CSS** – Custom styling

## 🚀 How to Run Locally

1️⃣ Clone the repository:

```bash
git clone https://github.com/sru2808/stock-market-dashboard.git
cd stock-market-dashboard

2️⃣ Install dependencies:

pip install -r requirements.txt

3️⃣ Run the app:

streamlit run app.py

4️⃣ Open your browser at: http://localhost:8501
