📝 Project Overview

The Stock Market Dashboard is an interactive web application built with Streamlit. It helps users:

Track real-time stock prices
Manage and monitor a personal portfolio
Visualize stock trends with interactive candlestick charts
Read the latest financial and market news

This project is ideal for students, beginners, and investors who want a visual and user-friendly tool to monitor the stock market.

💡 Features

Market Overview

View latest, highest, and lowest prices of any stock
Interactive candlestick charts to track stock trends
Next-day price prediction for insights

Portfolio Tracker

Add your stocks with number of shares
Track portfolio value and investment performance
Clean data table view

News Section

Get the latest financial news
Clickable links to full articles for deeper insights

Profile Section

Displays user information and project details

Dark Mode & Stylish UI

Modern dark theme
Custom buttons, input fields, and cards
⚙️ Technologies Used
Python 3.x
Streamlit – Web app framework
Plotly – Candlestick and trend charts
Pandas – Data handling and analysis
YFinance / APIs – Real stock data (optional)
HTML & CSS – Custom styling
🚀 How to Run Locally
Clone the repository:
git clone https://github.com/sru2808/stock-market-dashboard.git
cd stock-market-dashboard
Install dependencies:
pip install -r requirements.txt
Run the app:
streamlit run app.py
Open your browser at http://localhost:8501
