📝 About the Project

The Stock Market Dashboard is an interactive web application built using Streamlit. It allows users to:

Track stock prices in real-time or simulated data
Manage and monitor their personal portfolio
Visualize stock trends with candlestick charts
Stay updated with the latest financial news

It is designed for students, beginners, and investors who want an easy and visual way to monitor the stock market.

💡 Features
📈 Market Overview – Latest, high, and low prices, interactive charts, next-day price prediction
💼 Portfolio Tracker – Add stocks, view portfolio value, and track your investments
📰 News Section – Read the latest market news with clickable links to full articles
👤 Profile Section – Display personal info and project details
🌙 Dark Theme UI – Stylish dark mode with custom input boxes, buttons, and cards
⚙️ Technologies Used
Python 3.x
Streamlit – Web application framework
Plotly – Charts and candlestick visualization
Pandas – Data handling and analysis
YFinance / APIs – Real stock market data (optional)
HTML & CSS – Custom styling
🚀 How to Run Locally
Clone the repository:
git clone https://github.com/sru2808/stock-market-dashboard.git
Install dependencies:
pip install -r requirements.txt
Run the app:
streamlit run app.py
Open your browser at: http://localhost:8501
